# My Portfolio

Single-page portfolio built with vanilla HTML/CSS/JS.

## Features

- Anime.js micro-interactions
- particles.js background (`particlejs-config.json`)
- Skills playground (floating / bouncing icons)
- Contact form powered by EmailJS (no backend)

## Run locally

This is a static site. For best results, use a local server.
Open `http://localhost:5500/`.

## Structure
- `index.html`
- `css/`
- `js/`
- `assets/`
